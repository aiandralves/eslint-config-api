# Aiandra ESLint config

## Whats included?

-   Standard config base;
-   JSX a11y plugin;
-   Prettier;

## Setup

Install dependencies:

```
npm i -D @aiandralves/eslint-config
```

Inside `.eslintrc.json`

### Node.js

```
{
    "extends": [
        "@aiandralves/eslint-config/node"
    ]
}
```

### Angular

```
{
    "extends": [
        "@aiandralves/eslint-config/angular"
    ]
}
```
